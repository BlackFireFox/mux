# MUX
music player/downloader
***
### Installation
`sudo make install`
### Uninstallation
`sudo make uninstall`
### Reinstallation
`sudo make reinstall`
# Usage
<ul>
<li>"<i><b>s</b></i>" for new search</li>
<li>"<i><b>&#60;</b></i>" for page+1</li>
<li>"<i><b>&#62;</b></i>" for page-1</li>
<li>"<i><b>num</b></i>" for play num</li>
<li>"<i><b>d num</b></i>" for download num</li>
</ul>
